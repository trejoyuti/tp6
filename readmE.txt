TP6 - Tabla Informativa con HTML y CSS

# [cite_start]2. Descripción del Proyecto [cite: 42]
Este proyecto es el Trabajo Práctico N°6 de la materia Laboratorio de Programación. El objetivo es desarrollar una página web que muestre una tabla informativa utilizando HTML5 semántico y CSS3.

## Temática Elegida
La temática elegida es un **"Horario Escolar"** para el curso 6° G.

## Justificación
Elegí esta temática porque es un ejemplo claro y práctico del uso de tablas. [cite_start]Permite aplicar de forma natural los requisitos técnicos obligatorios, como `colspan` (utilizado para unificar las celdas durante el receso) y `rowspan` (utilizado para materias que ocupan más de un bloque horario, como Educación Física)[cite: 27].

# [cite_start]3. Tecnologías Usadas [cite: 44]
* [cite_start]**HTML5**: Para la estructura semántica del documento (`header`, `main`, `footer`) [cite: 28] [cite_start]y la tabla (`table`, `thead`, `tbody`, `caption`, etc.)[cite: 20, 21].
* [cite_start]**CSS3**: Para los estilos visuales, aplicando `border-collapse` [cite: 32][cite_start], colores a encabezados, filas alternadas (`nth-child`) [cite: 33] [cite_start]y el uso de una fuente externa de Google Fonts (Lato)[cite: 34].

# [cite_start]4. Estructura del Proyecto [cite: 46]
El repositorio contiene los siguientes archivos:

* `index.html`: Archivo principal que contiene la estructura HTML de la página web.
* [cite_start]`styles.css`: Hoja de estilos externa [cite: 31] que define la apariencia visual de la tabla y la página.
* `README.md`: Este mismo archivo, que contiene la documentación del proyecto.
